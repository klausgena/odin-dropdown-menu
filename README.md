# ns-dropdown-menu

A simple javascript activated dropdown menu

## Usage

Create a dropdown menu in html with the following structure:

`<nav class="dropper">
<button class="drop-btn">Menu</button>

<ul class="drops hidden">
<li><a href="#">One</a></li>
<li><a href="#">Two</a></li>
<li><a href="#">Three</a></li>
<li><a href="#">Four</a></li>
<li><a href="#">Five</a></li>
</ul>
</nav>`
